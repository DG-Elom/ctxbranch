# /tree

Show the current branch tree, with status markers, intent tags, and the current
branch highlighted.

## Usage

```
/tree
```

## Output example

```
my-project
├── main ● current
│   ├── digression-jwt-1776808900 <digression> (merged)
│   └── hypothesis-cache-1776809000 <hypothesis> (thrown)
```

## Under the hood

!`ctxbranch tree`
